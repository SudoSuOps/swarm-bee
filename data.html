API Documentation
Three endpoints. JSON responses. CORS enabled.
GET — Public
/api/data/catalog
Returns the full specialty catalog with pair counts, pricing tiers, and metadata.
curl https://swarmandbee.com/api/data/catalog
GET — Public
/api/data/sample?specialty=neurology
Returns 10 random platinum pairs. Optional specialty filter. No authentication required.
curl https://swarmandbee.com/api/data/sample
curl "https://swarmandbee.com/api/data/sample?specialty=surgery"
GET — API Key Required
/api/data/pull?specialty=surgery&limit=100&offset=0
Paginated access to platinum pairs by specialty. Max 1,000 pairs per request. Requires Authorization: Bearer sk_swarm_xxx header.
curl -H "Authorization: Bearer YOUR_API_KEY" \
  "https://swarmandbee.com/api/data/pull?specialty=cardiology&limit=50&offset=0"

Response format: All endpoints return JSON with ok, count, pairs[], and pagination metadata (total, offset, has_more).
